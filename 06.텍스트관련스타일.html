텍스트 관련 스타일

color : 텍스트의 색상 지정

        color : 색상명 | 16진수 | rgb(x,x,x) | rgba(x,x,x,x) | hsl(x,x,x) | hsla(x,x,x,x)

색상 지정하기

text-decoration : 텍스트에 줄을 긋거나 없앨 때 사용

        text-decoration : none | underline | overline | line-through
  • 텍스트 영역 위에 줄 긋기
  • 텍스트 영역 중간에 줄 긋기
  • 텍스트 영역 아래에 줄긋기
네이버로 이동

text-transform : 영문텍스트의 대소문자 변환시 사용

        선택자{
            text-transform : uppercase | lowercase | capitalize;
        }
  1. uppercase : 모든 영문자를 대문자로
  2. LOWERCASE : 모든 영문자를 소문자로
  3. capitalize : 영문자의 첫 글자만 대문자로

text-shadow : 텍스트에 그림자 효과를 줄 때 사용

        선택자{
            text-shadow: 가로거리(x), 세로거리(y) [번짐정도] [그림자색상];
        }
